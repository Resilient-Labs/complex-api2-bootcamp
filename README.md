### Description
Welcome to the "å­¦è‹±è¯­ - Learn English" app! This web application is designed to assist Chinese speakers in learning English. Simply enter an English word, click on the search button, and obtain the pronunciation, Chinese translation, and part of speech of the word. The part of speech is presented in both English and Chinese.

The app integrates data from two APIs:
1. <a href="https://www.wordsapi.com/">WordsAPI</a>: Provides pronunciation and part of speech.
2. <a href="https://cloud.google.com/translate">Google Translation API</a>: Uses the part of speech from API #1 to fetch the Chinese part of speech, alongside the Chinese translation of the word.
